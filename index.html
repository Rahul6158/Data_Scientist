Data Science Fundamentals

What is Data Science?

Data Science is an interdisciplinary field that uses scientific methods, algorithms, and systems to extract knowledge and insights from structured and unstructured data. It combines:
  • Statistics and mathematics
  • Programming and data engineering
  • Domain expertise
  • Data visualization
  • Machine learning

What is the core of Data Science?

The core revolves around:
  • Data-driven decision making: Using evidence from data rather than intuition
  • Extracting meaningful patterns: Finding signals in noise
  • Creating predictive models: Forecasting future outcomes
  • Communicating insights: Telling compelling stories with data

How does Data Science differ from related fields?

  • Business Intelligence: Focuses on descriptive analytics (what happened)
  • Statistics: Theoretical foundation without implementation
  • Machine Learning: Subset focused on predictive modeling
  • Data Engineering: Focuses on data pipelines and infrastructure
Data Science integrates aspects of all these fields.

Why is Data Science in demand?

Key drivers include:
  • Explosion of data generation (2.5 quintillion bytes daily)
  • Cheaper storage and processing power
  • Proven business value across industries
  • Shortage of skilled professionals
  • Competitive advantage for companies

What problems can Data Science solve?

Real-world applications:
  • Healthcare: Disease prediction, drug discovery
  • Finance: Fraud detection, risk assessment
  • Retail: Recommendation systems, inventory optimization
  • Manufacturing: Predictive maintenance
  • Marketing: Customer segmentation, churn prediction

Key skills for Data Scientists?

Technical:
  • Programming (Python/R)
  • SQL and databases
  • Machine learning
  • Data visualization
Soft Skills:
  • Communication
  • Business acumen
  • Problem-solving
  • Curiosity

Essential programming languages?

  • Python (80%+ usage): General-purpose with rich libraries
  • R: Strong in statistical analysis
  • SQL: Essential for data extraction
  • Bonus: Scala for big data, JavaScript for visualization
Python is recommended for beginners due to its versatility.

Essential libraries/frameworks?

  • Data Manipulation: Pandas, NumPy
  • Visualization: Matplotlib, Seaborn, Plotly
  • Machine Learning: Scikit-learn, XGBoost
  • Deep Learning: TensorFlow, PyTorch
  • Big Data: Spark, Dask

Types of Machine Learning?

  • Supervised: Labeled data (classification, regression)
  • Unsupervised: Unlabeled data (clustering, dimensionality reduction)
  • Reinforcement: Reward-based learning (robotics, game AI)
  • Semi-supervised: Mix of labeled and unlabeled data

Career and Learning Path

Typical Data Science roles?

  • Data Analyst: Reports and dashboards
  • Data Scientist: Modeling and advanced analytics
  • ML Engineer: Production models and systems
  • Data Engineer: Pipelines and infrastructure
  • BI Developer: Business reporting

Day-to-day responsibilities?

Common tasks:
  • Data cleaning and preprocessing (60% time)
  • Exploratory data analysis
  • Model development and evaluation
  • Creating visualizations and reports
  • Collaborating with stakeholders
  • Deploying and monitoring models

Educational background?

Common paths:
  • Formal: Degrees in CS, Stats, Math, or related fields
  • Bootcamps: Intensive 3-6 month programs
  • Self-taught: Online courses and projects
  • Transitioning: From related fields like engineering or economics
Many successful data scientists come from non-traditional backgrounds.

Typical career path?

Common progression:
  1. Data Analyst (0-2 years)
  2. Junior Data Scientist (2-3 years)
  3. Data Scientist (3-5 years)
  4. Senior Data Scientist (5+ years)
  5. Lead/Manager or Specialist roles
Many branch into ML Engineering, Analytics Leadership, or AI Research.

Time to learn fundamentals?

Varies by background:
  • Technical background: 3-6 months full-time
  • Non-technical background: 6-12 months
  • Part-time learning: 12-18 months
Focus on consistent, project-based learning rather than time estimates.

Foundational Statistical Concepts

What is Statistics, and why is it a fundamental pillar of Data Science?

Statistics is the science of collecting, analyzing, interpreting, and presenting data. It's fundamental to Data Science because:
  • Provides methods for data analysis and inference
  • Helps quantify uncertainty in conclusions
  • Forms the mathematical foundation for machine learning
  • Enables valid conclusions from incomplete data (samples)

What are the different types of data?

  • Nominal: Categories without order (e.g., colors)
  • Ordinal: Ordered categories (e.g., survey scales)
  • Interval: Numerical with consistent intervals (e.g., temperature in ¬∞C)
  • Ratio: Numerical with true zero (e.g., height, weight)
Understanding types is crucial because they determine which statistical methods and visualizations are appropriate.

What are descriptive statistics?

Descriptive statistics summarize and describe features of a dataset:
  • Central tendency: Mean, median, mode
  • Dispersion: Range, variance, standard deviation
  • Shape: Skewness, kurtosis
Example: The average (mean) test score was 75 with a standard deviation of 10.

What are the different types of distributions?

Key distributions in statistical modeling:
  • Normal: Bell curve, many natural phenomena
  • Binomial: Success/failure outcomes
  • Poisson: Counts of rare events
  • Exponential: Time between events
Important because many statistical tests assume specific distributions.

What is probability?

Probability quantifies how likely an event is to occur (0 = impossible, 1 = certain). It underpins statistical inference by:
  • Modeling uncertainty in conclusions
  • Calculating confidence intervals
  • Determining p-values in hypothesis testing
  • Forming Bayesian approaches

What are random variables?

Random variables are numerical descriptions of uncertain outcomes:
  • Discrete: Countable outcomes (e.g., dice rolls)
  • Continuous: Measurable outcomes (e.g., temperature)
They form the building blocks for probability distributions and statistical models.

Inferential Statistics & Hypothesis Testing

What is inferential statistics?

Methods for drawing conclusions about populations from sample data:
  • Estimation (point and interval)
  • Hypothesis testing
  • Regression analysis
  • ANOVA
Example: Using a sample survey to estimate political preferences nationwide.

What are confidence intervals?

A range of values that likely contains a population parameter:
  • 95% CI means 95% of such intervals will contain the true parameter
  • Calculated as: estimate ¬± margin of error
  • Wider intervals indicate more uncertainty
Interpretation: "We're 95% confident the true mean is between X and Y."

  • Type II (False Negative): Failing to reject false H‚ÇÄ
Trade-off: Reducing one increases the other. Controlled via:
  • Significance level (Œ±) for Type I
  • Sample size and effect size for Type II

What are common statistical tests?

  • t-tests: Compare means (1-sample, 2-sample, paired)
  • Chi-squared: Test independence in categorical data
  • ANOVA: Compare means across >2 groups
  • Regression: Model relationships between variables
Choice depends on data type and research question.

Vector Fundamentals: Flash Cards

1 Three Perspectives on Vectors

Physics Student View

How would you describe a vector in physical space?

  • Arrows pointing in space
  • Defined by length and direction
  • Can move anywhere (translation invariant)
  • 2D (plane) or 3D (space) vectors

Computer Science View

How is a vector represented in data analysis?

  • Ordered lists of numbers
  • Example: [square_footage, price]
  • Order matters in the list
  • Dimension = length of list

Mathematician's View

What's the abstract definition?

  • Anything with vector addition + scalar multiplication
  • Abstract algebraic structure
  • Generalizes both other perspectives
  • Basis for linear algebra theory
